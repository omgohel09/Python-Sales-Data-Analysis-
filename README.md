# Python-Sales-Data-Analysis🛒 E-Commerce Sales Insights Dashboard using Python

This project performs an end-to-end Exploratory Data Analysis (EDA) on an E-Commerce dataset to uncover patterns, trends, and actionable business insights. Using Python, Pandas, NumPy, Matplotlib, Seaborn, and Plotly, the notebook visualizes KPIs such as sales, profit, customer behavior, and product performance.

⭐ Project Goals

• Identify top-performing product categories and sub-categories. • Analyze state and regional contributions to overall sales and profit. • Understand monthly and yearly sales trends. • Determine whether higher discounts reduce profitability. • Evaluate performance across customer segments. • Highlight top customers and best-selling products. • Compare sales and profit across shipping modes.

🧠 Skills Demonstrated

• Data Cleaning & Preparation using Pandas • Exploratory Data Analysis (EDA) • Time Series Trend Analysis • Aggregations & Groupby Operations • Insightful Data Visualizations (Matplotlib, Seaborn, Plotly) • Business Insights & Storytelling • Notebook-style Dashboard Presentation

📂 Project Structure

• Data Loading & Cleaning • Feature Engineering (extracting Year, Month, etc.) • Time Series Analysis • Monthly Sales Trend • Yearly Sales Trend • Category & Sub-Category Analysis • State & Regional Performance Analysis • Customer Segment Analysis • Discount vs Profit Analysis • Top Customer Identification • Top Products by Sales & Profit • Shipping Mode Comparison

📌 Key Insights :

• Sales and Profit are uneven across states, with a few states contributing the majority of revenue. • Technology and Office Supplies categories show strong sales performance. • Some sub-categories generate high sales but low or negative profit due to heavy discounting. • Higher discounts generally reduce profit margins, indicating poor discount strategy. • Consumer segment performs the best in both sales and order volume. • Shipping mode usage varies, but Standard Class is the most commonly selected option. • Top customers and products follow a Pareto pattern, where a smaller percentage contributes heavily to total sales.

🛠️ Technologies Used

• Python • Pandas • NumPy • Matplotlib • Seaborn • Plotly • Jupyter Notebook

🚀 How to Run the Project

• Clone the repository • Install required libraries • Open the Jupyter Notebook • Run all cells to generate insights and plots

✅ Conclusion

This project provides a complete overview of an e-commerce business’s sales performance. It covers time trends, customer behavior, discount impact, and product performance, offering valuable insights that can help improve business decisions.
